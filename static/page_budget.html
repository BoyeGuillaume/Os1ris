Writing Queries

Available variable (Value)

These are the variable available for usage when writing budget for expenses

  • wealth — (number) total money currently variable within the bank account minus safety level (sum of all transaction previously matched by the income minus expense)
  • raw_wealth — (number) total money available (sum of all transaction previously matched by the income minus expense)
  • total_income — (number) sum of all incomes during the interval minus safety level
  • raw_income — (number) sum of all raw incomes (should prefer total_income)

Available variable (Filter)

These are the variable available for usage when writing filter for Incomes and Expenses

  • id (integer) — the unique index corresponding with the transaction
  • account (string) — name of the account where the transaction occurred
  • amount (number) — amount of the transaction
  • date (date) — date at which the transaction occurred
  • available_date (date) — date at which the transaction become effective within the bank system
  • registration_date (date) — date at which the transaction was registered within the system
  • description (string) — the description corresponding with the transaction
  • tags (string) — list of tags associated with a transaction, separated with ,
